## Sobre o Projeto

Sistema de pedidos online para a rede de lanchonetes **Raízes do Nordeste**, desenvolvido como projeto multidisciplinar da UNINTER (trilha Front-end).

O sistema simula uma experiência completa de pedidos, desde a visualização do cardápio até a avaliação do pedido, passando por carrinho, cupons, pontos de fidelidade e pagamento integrado com serviço externo.

## Funcionalidades

- **Login/Cadastro** - Usuários podem criar conta ou acessar com e-mail/senha
- **Modo Visitante** - Visualizar cardápio sem necessidade de login
- **Múltiplas Unidades** - Cardápios diferentes para Recife, Olinda e Caruaru
- **Cardápio com Imagens** - Produtos com foto, descrição e preço
- **Busca de Produtos** - Filtro em tempo real por nome ou descrição
- **Carrinho de Compras** - Adicionar/remover itens, ajustar quantidades
- **Cupons de Desconto** - Códigos: FIDELIDADE10 (10%), FIDELIDADE20 (20%), BEMVINDO5 (R$5)
- **Pontos de Fidelidade** - Acumula 10% do valor do pedido em pontos (10 pontos = R$1)
- **Forma de Entrega** - Retirada na unidade ou delivery (com campo de endereço)
- **Pagamento Desacoplado** - Simulação de serviço externo (Crédito, Débito, PIX, Dinheiro)
- **Status do Pedido** - Barra de progresso: Pagamento → Recebido → Preparando → Pronto → Retirado
- **Avaliação** - Nota de 1 a 5 estrelas + comentário opcional
- **LGPD** - Consentimento explícito para uso de dados pessoais
- **Responsivo** - Funciona em desktop, tablet e mobile

## Tecnologias Utilizadas

- **HTML5** - Estrutura das telas
- **CSS3** - Estilização, grid responsivo e animações
- **JavaScript (Vanilla)** - Toda a lógica do sistema (sem frameworks)
- **LocalStorage** - Persistência de dados (usuários, carrinho, sessão)
- **GitHub Pages** - Hospedagem do protótipo
